# GO-Gin-REST-API

## Features

- Auto reload using AIR.
- All REST APIs (GET, POST, PUT, DELETE)

## Start server

```bash
air run main.go
```

### Add new user

**URL:** POST localhost:8080/addUser
Body

```json
{
  "id": "7",
  "name": "Gyanendra Knojiya",
  "userName": "Gyan",
  "email": "gyanendrak064@gmail.com",
  "phone": "+918802879231",
  "website": "gyanendra.tech"
}
```
